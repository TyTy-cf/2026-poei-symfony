
# 2026 POEI Symfony

## Installation

Commencez par vous créer une branche, puis :

Allez dans le fichier `docker/Dockerfile`, modifier les lignes :

```
ARG USER_ID=1001
ARG GROUP_ID=1001
```

Par votre id (1000)


Lancer la commande `make init`

## Helper

URL Site : https://localhost:8443/
PhpMyAdmin : http://localhost:8080/
